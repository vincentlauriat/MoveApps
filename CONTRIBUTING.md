# Contribuer

MoveApps est un projet personnel de Vincent Lauriat, développé en solo pour ses propres besoins de migration de projets entre `~/DevApps` et `~/Documents/GitHub`. Le dépôt est public par souci de transparence et pour que l'auto-update (Sparkle) fonctionne, pas pour recruter des contributeurs actifs.

## Issues

Les rapports de bug et suggestions sont bienvenus via les [issues GitHub](https://github.com/vincentlauriat/MoveApps/issues). Merci de décrire le comportement observé, le comportement attendu, et si possible les étapes pour reproduire.

## Pull requests

Ce projet n'est pas activement ouvert aux contributions externes : les choix de conception et le rythme de développement suivent les besoins personnels de Vincent. Une PR non sollicitée peut être ignorée ou fermée sans revue approfondie. Si vous avez une idée de correctif ou d'amélioration, ouvrir une issue au préalable pour en discuter est préférable à une PR directe.
